Primeiros Comandos SQL

SQL oferece comandos essenciais que permitem manipular e consultar dados em um banco de dados. Abaixo estão os mais usados:

  • SELECT – Seleciona dados de uma tabela.
  • INSERT INTO – Insere novos registros.
  • UPDATE – Atualiza dados existentes.
  • DELETE – Exclui registros.
  • CREATE TABLE – Cria novas tabelas.
  • DROP TABLE – Remove uma tabela.

Exemplo: SELECT

SELECT * FROM usuarios;

Retorna todos os registros da tabela usuarios.

Exemplo: INSERT

INSERT INTO usuarios (nome, email)
VALUES ('João Silva', 'joao@email.com');

Exemplo: UPDATE

UPDATE usuarios
SET nome = 'João Pedro'
WHERE id = 1;

Exemplo: DELETE

DELETE FROM usuarios WHERE id = 1;
