Exemplos Avançados

Aqui estão alguns exemplos avançados que envolvem JOINs, GROUP BY, funções e subconsultas.

JOIN entre tabelas

SELECT usuarios.nome, pedidos.data
FROM usuarios
JOIN pedidos ON usuarios.id = pedidos.usuario_id;

GROUP BY com funções agregadas

SELECT categoria, COUNT(*) AS total_produtos
FROM produtos
GROUP BY categoria;

Subconsulta

SELECT nome
FROM usuarios
WHERE id IN (SELECT usuario_id FROM pedidos);

Ordenação com ORDER BY

SELECT nome, idade
FROM usuarios
ORDER BY idade DESC;

Limitar resultados com LIMIT

SELECT * FROM produtos
LIMIT 5;
